=== 1. BASE DE DATOS (PostgreSQL con Docker) ===
Para encender la base de datos (se ejecuta en segundo plano):
sudo docker compose -f /home/gonza/code/cafeteria-gestion/docker-compose.yml up -d db

Para apagar la base de datos:
sudo docker compose -f /home/gonza/code/cafeteria-gestion/docker-compose.yml down


=== 2. BACKEND (FastAPI - Python) ===
Para encender el backend, abre una terminal, navega a la carpeta backend y ejecuta:
cd /home/gonza/code/cafeteria-gestion/backend
source venv/bin/activate
PYTHONPATH=.. uvicorn main:app --reload

Para apagar el backend:
Presiona Ctrl + C en esa misma terminal.


=== 3. FRONTEND (React - Vite) ===
Para encender el frontend, abre OTRA terminal, navega a la carpeta frontend y ejecuta:
cd /home/gonza/code/cafeteria-gestion/frontend
npm run dev

Para apagar el frontend:
Presiona Ctrl + C en esa misma terminal.
